# 📝 Python To-Do List Application

## 📌 Project Overview

This project is a simple command-line To-Do List application developed
in Python as part of my internship at **DecodeLabs**.

The main objective of this project is to practice Python fundamentals by
building a real-world application using lists, loops, conditional
statements, functions, and user input.

------------------------------------------------------------------------

## 🚀 Features

-   Add new tasks
-   View all tasks
-   Delete existing tasks
-   Update existing tasks
-   Interactive menu-driven interface
-   Runs continuously until the user exits

------------------------------------------------------------------------

## 🛠 Technologies Used

-   Python 3
-   Lists
-   Functions
-   Match-Case
-   While Loop
-   For Loop
-   Conditional Statements
